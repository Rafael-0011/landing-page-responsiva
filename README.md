# Landing Page Responsiva

Exemplo de landing page responsiva desenvolvida apenas com HTML e CSS. Inclui layout adaptável, tipografia e botões estilizados.

## Sobre
Este repositório contém uma landing page construída com foco em responsividade e boas práticas de estrutura HTML e organização de CSS. O objetivo é servir como exemplo simples para aprendizado e como base para projetos pequenos.

## Demonstração
- Abra o arquivo `index.html` no navegador para ver a página localmente.
- Para uma experiência mais próxima de produção, você pode servir a pasta com um servidor HTTP simples, por exemplo:
  - Python 3: `python -m http.server 8000`
  - Node.js (serve): `npx serve .`

## Tecnologias
- HTML (≈54%)
- CSS (≈46%)
Nenhuma dependência ou build step é necessário — apenas um navegador moderno.

## Funcionalidades
- Layout responsivo usando Flexbox / CSS Grid (conforme implementado)
- Tipografia e cores definidas em variáveis CSS
- Botões e formulários estilizados
- Estrutura semântica de HTML para melhor acessibilidade e SEO básico

## Estrutura do repositório
- `index.html` — página principal (landing)
- `css/` — arquivos de estilo (ex.: `styles.css`)
- `assets/` — imagens e recursos estáticos (se houver)
- `README.md` — documentação (este arquivo)
